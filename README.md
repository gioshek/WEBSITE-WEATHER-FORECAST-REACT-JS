# Инструкция для запуска проекта
1. `npm i`
2. `npm run build`
3. `npm run start`

# О чём проект
Данный вебсайт был создан на основе библиотеки React с использованием компонентов, а также адаптирован с помощью библиотеки стилей Bootstrap. На нём мы можем смотреть прогноз погоды интересующего нам города, благодаря запросам с стороннего API, в данном случае WEATHER API.
